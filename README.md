## SETUP INSTRUCTIONS

Follow these steps to get the app running locally:

### PREREQUISITES

Make sure you have the following installed:

- Node.js
- npm
- Git

### INSTALLATION

1. Clone the repository

2. Install Dependencies 
	- npm install
	- yarn install

3. Start development server
	- npm run dev
	- yarn start

4. Open the app
	- Visit localhost in your browser.
